Mail vs Thread

为什么单封邮件页仍然需要存在

  • 只有这层适合看完整 envelope、header、delivery map。
  • Thread 页适合看上下文，不适合承载原始邮件真相。
  • Mailbox 页适合看完整活动，不适合展开每封邮件的细节。

Downstream

进入 Mail Detail 后要看到的东西

  • 完整 subject、body、from/to/cc。
  • `in_reply_to`、`references`、所属 thread。
  • delivery 状态、linked resources、related tasks。
